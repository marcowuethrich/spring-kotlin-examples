# Spring Boot Kotlin Examples

This little project shows some examples how to use the spring project with kotlin.


### Projects:
- Spring cloud config server/client
- Spring cloud service discovery server/client
- Spring actuator
- Spring sleuth
- Spring openfeign
- Spring gateway
